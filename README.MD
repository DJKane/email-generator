# Email Generator for Kane Engineering

A Node.js application that generates professional, friendly client emails using the Claude API. Built to develop a consistent, approachable communication voice for Kane Engineering Group Inc.

## Features

- **Interactive email generation** - Answer a few questions, get a polished email
- **Custom voice guidelines** - Warm, approachable, professional tone
- **Multiple email types** - Status updates, proposals, problem-solving, follow-ups
- **Claude API integration** - Uses Claude Opus 4.6 for high-quality output

## Project Purpose

This is a learning project for:
- Claude API integration with Node.js
- Building Claude Code workflows
- Developing templated communication patterns
- Integration with Obsidian for template management

## Setup

### Prerequisites
- Node.js v25.2.1 or higher
- Anthropic API key (get one free at https://console.anthropic.com/keys)

### Installation

```bash
git clone https://github.com/YOUR-USERNAME/email-generator.git
cd email-generator
npm install
```
